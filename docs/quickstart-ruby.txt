Ruby Quickstart
===============

An Example using Sinatra
------------------------

Inside your developer dashboard there is a form for you to enter your callback location.  
That full url must point to a location that takes a post and processes the SAML response.  
In Sinatra that would look like this:

    :::ruby
    require 'rubygems'
    require 'ruby-saml'
    require 'sinatra/base'
    class Application < Sinatra::Base
      
      post '/login' do
        settings = Onelogin::Saml::Settings.new
        settings.assertion_consumer_service_url = "" #your callback url
        settings.idp_cert_fingerprint = "e019441a82f1f2ebfc42397aea85b7b5f398562c"
        settings.name_identifier_format = "urn:oasis:names:tc:SAML:2.0:nameid-format:persistent"
        response          = Onelogin::Saml::Response.new(params[:SAMLResponse])
        response.settings = settings
        session[:user_id] = response.name_id
        # ...
      end
    end
